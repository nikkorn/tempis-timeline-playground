Featured

🎮 Playground

Interactive testing environment with comprehensive controls for all timeline options. Adjust layout, styling, data, and behavior in real-time.

📡 Crypto Price Stream

Live cryptocurrency price stream via WebSocket demonstrating real-time data updates. Shows 10 cryptocurrencies with color-coded price movements.

🏭 Production Downtime

Production line downtime tracking with detailed event information. Click any downtime event to see an animated details panel with cause, resolution, cost, and notes.

🔄 Lazy Loading

Server request logs loaded on demand as you pan and zoom. Demonstrates onRangeChange callback with debounced data fetching and caching.

💬 Custom Tooltips

Project sprint tracker with rich custom tooltip cards. Hover any task to see assignee details, status, priority, and a live progress bar.

🌍 Timezone Explorer

Global flight schedule rendered with a Luxon date adapter. Switch timezones and watch the entire timeline re-render in different local times.

📊 Server Monitoring

Weekly Kubernetes cluster overview with timeline bands highlighting maintenance windows, off-peak hours, deployment markers, and incident lines.

🌐 RTL Project Schedule

Arabic mobile app development schedule with full right-to-left support. Toggle between RTL and LTR to see the timeline mirror its layout.

🎨 Color Palettes

Switch between curated color palettes and watch the timeline update instantly. Demonstrates the global palette API for theming.

📋 External Selection

Click rows in an event grid to select and focus items on the timeline. Demonstrates programmatic selection with setSelection() and focus().

📸 Export to Image

Music festival schedule across 5 stages over a weekend. Pick format, quality, and DPR then download the timeline as an image via toImage().

⌨️ Keypress Timeline

Interactive keyboard event visualization. Short presses create point-in-time items, long presses create range items that update in real-time.
